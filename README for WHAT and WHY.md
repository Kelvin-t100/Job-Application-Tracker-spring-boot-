# Job-Application-Tracker-spring-boot-
Job Application Tracker is a Spring Boot REST API built to help manage and organize job applications during the job search process.  Instead of manually maintaining notes or spreadsheets, this application provides a centralized way to store applications, search them, and update their progress throughout the hiring process.


##WHY I BUILT THIS??
While applying to more than 60 software development roles, I found it difficult to keep track of each application, its current status, and personal notes. Rather than using spreadsheets or note-taking apps, I built this REST API to solve that problem while strengthening my backend development skills.
